# Número Mágico
Juego del Número Mágico WEB

* El juego generará automaticamente un número entre el 1 y 1000.
* Como usuario, deberás intentar adivinar este número, tienes 10 intentos para ello.
* El juego te dará pistas sobre el número a adivinar y podrás ver los números que has ingresado para no repetirlos.
* También tendrás una lista con los registros de tus partidas.
